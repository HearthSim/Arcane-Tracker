Arcane Tracker privacy policy

 

Arcane Tracker is a tracker for Hearthstone on Android. It uses the Hearthstone log files generated by the game and stored on your phone to help you improve your gameplay. By default Arcane tracker does not require any account and does not collect any personal data.

You can opt-in third party services like trackobot.com or hsreplay.net in which case your game data is directly sent to the respective services without transiting by the Arcane Tracker servers. See 'Third party services' for more information.

Some non-personal technical data is collected to improve the quality and performance of the Arcane Tracker software. See 'Technical data' for more information

Data stored on your device

Hearthstone logs are read by Arcane Tracker to analyze the games you play. These files contain your battle.net tag as well as your opponent's and the details of the cards played during a game. This information is used to build the overlay as well as your win/loss count and game history. This data is stored in a database on your device. Only you can access it and it is not transferred to the Arcane Tracker servers.

This includes:

  • Game history and statistics
  • Decks
  • Packs opened

Third party services

Trackobot.com

If you enabled Trackobot.com integration, after each game, Arcane Tracker will send a summary of the game to trackobot.com. This summary contains the class of hero you and your opponent played, the mode (ranked/casual/arena), if you had the coin, if you won, the date of the game, your rank (if applicable) and the list of card played every turn.

You can opt-out any time by going to Settings -> Track-o-bot -> unlink your account.

Hsreplay.net

If you enabled Hsreplay.net integration, after each game, Arcane Tracker will send the portion of log file containing the game to hsreplay.net. The Hearthstone log file contains contains your battle.net tag as well as your opponent's and the details of the cards played during a game. You can find an example here

You can check HSreplay.net privacy policy on here.

You can opt-out any time by going to Settings -> Hsreplay -> disable.

Technical data

Firebase analytics

Arcane Tracker uses Firebase Analytics to get insights about how the app is used. This allows us to spend more time on areas that are used the most, make the least used features more discoverable and more generally improve the user experience. Typical events we monitor are the number of arena games played vs ladder games, number of hsreplay/trackobot accounts enabled, number of times a given screen has been opened, etc... All this collected data is anonymous and contains no personal information.

Firebase crashlytics

Arcane Tracker uses Firebase Crashlytics to collect crash data and improve the application. All data collected is anonymous and contains no personal information.
